调试

1. G3367SDK初始化

Application中调用

>正式服

平台申请gameid

需经过sigGameID签名得到真实gameSigid

如：gameid为1；则真实id为GameSigIdUtils.sigGameID(this, “1”)

*平台会对包进行二次包，gameSigid将会修改，如果需要【校验登录】功能，开发需用gameSigid，而不是 gameid；

   /**         * 初始化         * @param context 上下文         * @param gameid  找平台申请id         * @param dxAppid  平台申请id         */		String gameSigid = GameSigIdUtils.sigGameID(this, gameid);		Game3367SDK.getInstance().init(this, gameSigid, dxAppid);

6. 提交角色信息

* 只有提交了角色信息，才可以支付

    /**     * 提交用户数据     *     * @param areaid   区服id     * @param roleid   角色id     * @param level    角色级别     * @param rolename //角色名称     */    private void submitUserData(String areaid, String roleid, int level, String rolename) {        UserOtherDate userDate = new UserOtherDate();            userDate.setAreaId(areaid);//必传            userDate.setRoleId(roleid);//必传            //  -------------            userDate.setRoleLevel(level);//等级必传            userDate.setRoleName(rolename);//角色名字必传            Game3367SDK.submitUserData(userDate, new GameListener() {                @Override                public void onSuccess(int code, Object response) {                    Log.d("submitUserData", "提交角色信息成功");                }                 @Override                public void onStart(int code) {                }                 @Override                public void onFailure(int code) {                }                 @Override                public void onError(String code) {                    Log.d("submitUserData", "提交角色信息失败");                }                 @Override                public void onCancel(int code) {                 }            });        }}
