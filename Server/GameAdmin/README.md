## 请求接口修改

```bash
# api for test dev
  文件: .env.development

# api for production environment
  文件: .env.production

```

##  安装项目

```bash
# install dependency
npm install
```
##  启动项目

```bash
npm run dev
```

This will automatically open http://localhost:9527

## 打包生成静态文件

```bash
# build for test dev
npm run build:dev

# build for production environment
npm run build:prod

生成的静态文件在 /dist 目录下
```
